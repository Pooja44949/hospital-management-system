# hospital-management-system
DBMS capstone project for managing patient records, doctors, appointments, and treatments

PROJECT FOLDER:
hospital_project/
│
├── app.py
├── templates/
│   ├── index.html
│   ├── add.html
│   └── edit.html


This project is a DBMS-based web application that manages:

- Patient records
- Doctor assignments
- Appointments
- Treatments

## Technologies Used
- Python (Flask)
- MySQL
- HTML/CSS

## Features
- Add / Edit / Delete patients
- Assign doctors
- Manage appointments
- View diagnosis
